# OPUS Design Studio Website

A modern, minimalist website for OPUS Design Studio. This website showcases our creative work and services with a clean, professional design inspired by industry leaders.

## Features

- Responsive design that works on all devices
- Modern, minimalist aesthetic
- Smooth animations and transitions
- Sections for Work, About, and Contact

## Structure

- `index.html` - Main homepage with all content sections
- `styles.css` - All styling and animations

## Getting Started

Simply open `index.html` in a web browser to view the website locally.

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Inter)
